Spatial Audio Accessibility Demo

Move the player toward the target using arrow keys or WASD. Distance sets loudness, left/right sets panning and inter-aural time difference (ITD), and up/down sets a low-pass filter. Each cue can be toggled on or off.

Keyboard help

Click or tab to the world, then:

  • Arrow keys or W A S D — move the player
  • O — announce offset and distance
  • B — announce bearing
  • V — announce raw audio values
  • 1 / 2 / 3 — toggle panning / ITD / low-pass
  • Enter or Space — start audio
